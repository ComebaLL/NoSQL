# NoSQL

## Постановка задачи

Создать датасет по прокату велосипедов.

## Карточка датасета bicycle_ranlt

### bicycles

поля объекта:

- id 
- name - наименование 
- type - тип велосипеда
- characteristics - характеристики:
    - frame size - размер рамы велосипеда
    - lifting capacity -  разрешенная масса


### clients

поля объекта:

- id
- first name -имя
- last name - фамилия
- phone - телефон


### orders

поля объекта:

- id
- client id - клиент id
- bicycle id - id велосипеда
- start -начало проката
- end - конец проката
- duration - длительность
- cost - цена
